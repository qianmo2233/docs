## Welcome to UltiTools-API

**UltiTools-API** is a modern framework for building Minecraft plugins on Spigot and Paper — inspired by Spring Boot–style development.

Build Minecraft plugins the way you'd build a backend application:

- Annotation-driven commands  
- IoC container with dependency injection  
- ORM-style data storage (MySQL / SQLite / JSON)  
- Scheduled tasks without `BukkitRunnable` boilerplate  
- Config validation & feature toggles  
- AOP transactions and exception handling  

Java 8+ compatible. Supports Minecraft 1.8 – 1.21.

## Why UltiTools?

Traditional Spigot development often means:

- Manual `onCommand()` switch blocks  
- Manual listener registration in `onEnable()`  
- Static singletons for shared services  
- Raw JDBC or file I/O  
- Repetitive `new BukkitRunnable()` tasks  

UltiTools eliminates boilerplate and lets you focus on business logic.

## Architecture Overview

UltiTools includes:

- **Command System** — Annotation-based command mapping  
- **Event System** — Auto-discovered listeners  
- **IoC Container** — Three-level cache with circular dependency resolution  
- **Query DSL** — Fluent query builder for all supported data stores  
- **AOP Proxy System** — CGLIB-based runtime interception  
- **Hot Module Loading** — Load/unload modules without restarting the server  
- **WebSocket + UltiPanel** — Remote server management
